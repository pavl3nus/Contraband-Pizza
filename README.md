# Сюжет
Вы устраиваетесь на работу доставщика пиццы в небольшом городке с одним лишь условием - не заглядывать в сумку с пиицей. Выполняете несколько заказов. После очередного заказа вы понимаете, что что-то здесь не так, потому что доставка пиццы превращается в полицейскую погоню. Оторвавшись от копов, вы решаете всё-таки заглянуть в сумку.
